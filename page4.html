Learning date objects in JavaScript

Writer: Rizky Syaiful Fattah

Date Objects

Disini kita akan belajar tentang date/tanggal. secara default, JavaScript akan menggunakan waktu yang ada di browser dan menampilkannya sebagai sebuah string,
contohnya: Sun Sep 08 2019 21:21:21 GMT +0700 (Western Indonesia Time)

Objek tanggal dibentuk dengan konstruktor new Date().
Berikut adalah 4 cara untuk membuat objek tanggal :

new Date()
new Date(year, month, day, hours, minutes, seconds, milliseconds)
new Date(milliseconds)
new Date(date string)

new Date()

contoh syntax objek tanggal diatas :

var d = new Date();
--> objek tanggal dengan tanggal dan waktu saat ini
hasilnya : Sun Sep 08 2019 21:21:21 GMT +0700 (Western Indonesia Time)

new Date(year, month, ...)

pada syntax new Date(year, month, day, hours, minutes, seconds, milliseconds) dapat diubah jumlah parameternya dari 6 hingga 2 (year, month).
penghitungan bulan dari 0-11 dengan januari 0 dan desember 11.

var d = new Date(2018, 11, 24, 10, 33, 30, 0);
--> year, month, day, hours, minutes, seconds, milliseconds
var d = new Date(2018, 11, 24, 10, 33, 30);
--> year, month, day, hours, minutes, seconds
var d = new Date(2018, 11, 24, 10, 33);
--> year, month, day, hours, minutes
var d = new Date(2018, 11, 24, 10);
--> year, month, day, hours
var d = new Date(2018, 11, 24);
--> year, month, day
var d = new Date(2018, 11);
--> year, month

new Date(dateString)

syntax ini membuat objek tanggal dari string tanggal yang diinput

var d = new Date("September 11, 2019 20:20:20");
hasilnya : Wed Sep 11 2019 20:20:20 GMT +0700 (Western Indonesia Time)

JavaScript menyimpan tanggal sebagai bentuk milisekon

bentuk milisekon ini sejak 1 Januari 1970, 00:00:00 UTC
zero time adalah January 01, 1970 00:00:00 UTC
satu hari adalah 86400000 milisekon

var d = new Date(10000000) --> parameter milisekon;
hasilnya Sat Mar 03 1973 16:46:40 GMT+0700 (Western Indonesia Time)

penanggalan berdasarkan abad waktu

satu dan dua digit tahun akan diinterpretasikan sebagai 19xx :

var d = new Date(99, 11, 24);
hasilnya : Fri Dec 24 1999 00:00:00 GMT+0700 (Western Indonesia Time)

menampilkan tanggalan

JavaScript secara default akan mengeluarkan tanggal dalam bentuk text format string :
Wed Mar 25 2015 07:00:00 GMT+0700 (Western Indonesia Time).
metode toString(), menampilkan objek tanggal dalam HTML diubah ke string
metode toUTCString(), mengubah tanggal ke string bentuk UTC
metode toDateString(), mengubah tanggal ke format yang mudah dibaca

d = new Date();
document.getElementById("demo").innerHTML = d.toString();
d = new Date();
document.getElementById("demo").innerHTML = d.toUTCString();
d = new Date();
document.getElementById("demo").innerHTML = d.toDateString();
