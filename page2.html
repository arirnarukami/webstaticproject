Math in Javascript

Writer: johanadi santoso

Math

Objek JavaScript Matematika memungkinkan untuk melakukan tugas matematika pada angka.

Contoh : 
Math.PI;
Math.PI akan menghasilkan nilai 3.141592653589793

Math.round()

Math.round (x) mengembalikan nilai x dibulatkan ke bilangan bulat terdekatnya(melakukan pembulatan): 

Math.round(4.7);
Hasilnya adalah 5. 
Math.round(4.4);
Hasilnya adalah 4.

Math.pow ()

Math.pow(x, y) mengembalikan hasil dari nilai x pangkat y: 

Math.pow(8, 2);
Hasilnya adalah 64.


Math.sqrt ()

Math.sqrt(x) mengembalikan akar kuadrat dari x: 

Math.sqrt(64);
Hasilnya adalah 8.


Math.abs ()

Math.abs(x) mengembalikan mutlak nilai (positif) dari x: 

Math.abs(-4.7);
Hasilnya adalah 4.7.


Math.ceil ()

Math.ceil(x) mengembalikan nilai x dibulatkan ke atas integer terdekatnya( Pembulatan ke atas ): 

Math.ceil(4.4);
Hasilnya adalah 5.


Math.floor ()

Math.floor(x)mengembalikan nilai x dibulatkan ke bawah integer terdekatnya( Pembulatan ke bawah ): 

Math.floor(4.7);
Hasilnya adalah 4.


Math.sin ()

Math.sin(x) mengembalikan sinus (nilai antara -1 dan 1) dari sudut x (diberikan dalam radian). Jika Anda ingin menggunakan derajat bukan radian, Anda harus mengkonversi derajat ke radian: Sudut dalam radian = sudut dalam derajat x PI / 180. 

Math.sin(90 * Math.PI / 180);
Kode diatas akan menghitung Sin dari 90 * ∏ /180, Hasilnya adalah 1 .


Math.cos ()

Math.cos(x) mengembalikan kosinus (nilai antara -1 dan 1) dari sudut x (diberikan dalam radian). Jika Anda ingin menggunakan derajat bukan radian, Anda harus mengkonversi derajat ke radian: Sudut dalam radian = sudut dalam derajat x PI / 180. 

Math.cos(0 * Math.PI / 180);
Kode diatas akan menghitung Cos dari 90 * ∏ /180, Hasilnya adalah 1 .


Math.min () dan Math.max ()

Math.min()dan Math.max()dapat digunakan untuk mencari nilai terendah atau tertinggi dalam daftar argumen: 

Math.min(0, 150, 30, 20, -8, -200);
Hasilnya adalah -200. 
Math.max(0, 150, 30, 20, -8, -200);
Hasilnya adalah 150.


Math.random ()

Math.random() mengembalikan nomor acak antara 0 (inklusif), dan 1 (eksklusif): 

Math.random();
Contoh: 

<!DOCTYPE html>
<html>
<body>

<p>Math.random() returns a random number between 0 and 1:</p>

<p id="demo"></p>

<script>
document.getElementById("demo").innerHTML = Math.random();
</script>

</body>
</html>

Matematika Properties (Konstanta))

JavaScript menyediakan 8 konstanta matematika yang dapat diakses dengan objek Math: 

Math.E			// mengembalikan nomor Euler
Math.PI 		// mengembalikan PI
Math.SQRT2 		// mengembalikan akar kuadrat dari 2
Math.SQRT1_2		// mengembalikan akar kuadrat dari 1/2
Math.LN2 		// mengembalikan logaritma natural 2
Math.LN10 		// mengembalikan logaritma natural dari 10
Math.LOG2E 		// mengembalikan basis 2 logaritma E
Math.LOG10E		// mengembalikan basis 10 logaritma E


matematika Constructor

Tidak seperti objek global lainnya, objek Math tidak memiliki konstruktor. Metode dan properti yang statis. Semua metode dan properti (konstanta) dapat digunakan tanpa menciptakan sebuah objek Math pertama.
