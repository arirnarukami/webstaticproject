Array in Javascript

Writer: M. Aryarafi Danendra

Array

Array adalah variabel khusus yang dapat menyimpan value lebih dari satu.

Syntax : 
var nama_array = [item1, item2,...];

Contoh dan cara-cara alternatif menuliskan array:

var motor = ["ducati", "r25", "ninja"]; 
var motor = [20,
	60,
	120
];

Mengakses Array

Cara mengakses elemen dalam array adalah dengan indeks. Pada array, indeks dimulai dari 0. 

var motor = cars[0];
var motor = ["ducati", "r25", "ninja"]; 
document.getElementById("demo").innerHTML = motor[0];
Hasilnya adalah ducati.

Cara mengakses array secara keseluruhan adalah sebagai berikut. 

var motor = ["ducati", "r25", "ninja"]; 
document.getElementById("demo").innerHTML = motor;


Mengubah Elemen pada Array

Cara mengubah elemen pada array adalah sebagai berikut. 

var motor = ["ducati", "r25", "ninja"]; 
motor[0] = "beat";
document.getElementById("demo").innerHTML = motor[0];


Looping Elemen Array

Cara looping elemen teraman dalam array adalah dengan for. 

var fruits, text, fLen, i;
fruits = ["Banana", "Orange", "Apple", "Mango"];
fLen = fruits.length;

text = "
    ";
for (i = 0; i < fLen; i++) {
  text += "
  • " + fruits[i] + "
    • ";
}
text += "
";

document.getElementById("demo").innerHTML = text;

Hasilnya adalah

  • Banana
  • Orange
  • Apple
  • Mango

Cara lainnya adalah dengan menggunakan Array.forEach().

var fruits, text;
fruits = ["Banana", "Orange", "Apple", "Mango"];

text = "
    ";
fruits.forEach(myFunction);
text += "
";

function myFunction(value) {
  text += "
  • " + value + "
    • ";
}


    Menambah Elemen Array

    Cara termudah menambah elemen ke array adalah dengan method push(). 

    var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.push("Lemon");

    Perbedaan Array dan Object

    Di JavaScript, array memakai index angka. Di JavaScript, object memakai index bernama.


    Kapan Pakai Array? Kapan Pakai Object?

    JavaScript tidak mendukung associative array. Object dipakai jika nama elemennya berupa string. Array dipakai jika nama elemennya berupa angka.
