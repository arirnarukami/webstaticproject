Random in Javascript

Writer: A. Ainun Herlambang

Random merupakan salah satu fasilitas yang sering digunakan untuk mendapatkan angka secara acak. Ada beberapa fungsi yang bisa digunakan sesuai kebutuhan.


Math.Random()

Digunakan untuk mendapatkan angka antara 0(inklusif) sampai 1(eksklusif).

Syntax : 
Math.random();

Math.random() selalu return nilai di bawah 1


Random Integer

Math.random() dengan Math.Floor dapat digunakan untuk return angka integer. 

Math.floor(Math.random() * 10);

Kode tersebut akan return nilai acak antara 0 sampai 9


Math.floor(Math.random() * 11);

Kode tersebut akan return nilai acak antara 0 sampai 10


Fungsi Acak yang Benar

Dari beberapa contoh di atas, ada cara yang biasa digunakan untuk return nilai acak sesuai kebutuhan.
Fungsi Javascript ini selalu return nilai acak antara min(included) dan max(excluded): 

function getRndInteger(min, max) {

return Math.floor(Math.random() * (max - min) ) + min;}

Sedangkan fungsi Javascript berikut akan return nilai acak antara min dan max(keduanya included): 
function getRndInteger(min, max) {

return Math.floor(Math.random() * (max - min + 1) ) + min;}
